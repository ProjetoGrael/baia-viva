# Project Memory

## Core
BaíaViva: monitoramento ambiental Baía de Guanabara RJ. Tema oceânico azul/verde.
Plus Jakarta Sans headings, Inter body. Zero budget: free tiers only.
Dados GPS em WGS84 (EPSG:4326). Exportação GeoJSON para ArcGIS.
Backend: Lovable Cloud (Supabase). Auth por email/senha. RLS em todas as tabelas.
Papéis: admin, professor, student, community. Roles em tabela separada.
